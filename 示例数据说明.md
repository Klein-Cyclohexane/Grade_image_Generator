# Excel文件格式示例

## 必需的列名

Excel文件必须包含以下8列，列名必须完全匹配：

| 列名 | 说明 | 示例 |
|------|------|------|
| 科目 | 课程名称 | 高等数学 I|
| 代号 | 课程代码 | GEMA1001 |
| 等级 | 成绩等级 | A+, A, A-, B+, B, B-, C+, C, C-, F, P, NP |
| 绩点 | GPA绩点（数值） | 4.0, 3.7, 3.3, 0.0（P/NP为0） |
| 学分 | 课程学分（数值） | 1, 2, 3, 4, 5 |
| 学科类型 | 课程类型 | 专业必修, 专业选修, 通选课, 全校必修 |
| 授课教师 | 教师姓名 | 张三 |
| 开课院系 | 院系名称 | IMS |

## 数据示例

```
科目,代号,等级,绩点,学分,学科类型,授课教师,开课院系
高等数学 I,GEMA1001,A+,4.0,4,自然通识,张三,IMS
线性代数,MATH1112,B+,3.7,4,自然通识,李四,IMS
计算机编程,CS100,B,3.0,4,专业必修,赵六,SIST
哲学导论B,GEHA1254,P,0.0,2,全校必修,孙七,IHS
```

## 注意事项

1. **等级格式**：必须使用标准格式（A+, A, A-, B+, B, B-, C+, C, C-, F, P, NP），大小写不敏感
2. **文件格式**：仅支持.xlsx格式的Excel文件
3. **数值格式**：绩点和学分必须是数字，不能是文本
4. **P/NP处理**：P和NP等级的课程通常绩点为0，不计入GPA计算
5. **空值处理**：如果某些字段为空，程序会自动处理，但建议填写完整
